CSS-BACKGROUNDS

1. background-color: (color); = tùy chỉnh màu sắc cho nền
opacity: (); = tùy chỉnh độ đậm của nền
background: rgba( , , , ) : tùy chỉnh cả 2
background: rgba(0, 128, 0, 0.3) /* Green background with 30% opacity */

2. background-image : url(link ảnh); set nền web bằng hình ảnh
background-repeat: repeat-( x, y, no-repeat); /* x: chiều ngang, y: chiều dọc , no-repeat: không lặp lại ảnh nền*/
background-position: (vị trí); vị trí đặt ảnh nền
background-attachment: chỉ định ảnh cố định hay cuộn theo trang
fixed : hiển thị cố định
scroll : cuộn theo web
